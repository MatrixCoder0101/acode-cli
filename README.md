# Acode Plugin CLI 

This is a Acode plugin cli tool to create plugins and easily work with it

## Installation

install it via npm 

```bash
npm install -g @coswat/acode-cli
```

## Usage

run it using `acode-cli` command

```bash
acode-cli --help
```
